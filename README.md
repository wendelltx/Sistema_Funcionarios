# Sistema de Cadastro de Funcionários

Este projeto consiste em um sistema de gerenciamento de pessoal desenvolvido em Java, permitindo o cadastro de colaboradores, controle de status (ativo/inativo) e cálculos automatizados de folha salarial por departamento.

### 🛠️ Tecnologias Utilizadas
* **Linguagem:** Java 21
* **IDE:** VS Code e Eclipse 
* **Controle de Versão:** Git & GitHub

### 🚀 Como Executar o Projeto
1. Clone o repositório:
   ```sh
   git clone [https://github.com/wendelltx/Sistema_Funcionarios.git](https://github.com/wendelltx/Sistema_Funcionarios.git)

   Acesse a pasta do projeto e compile os arquivos:

Bash
javac *.java

Execute o programa:

Bash
java TesteSistema

### 📂 Estrutura das Classes
* **Funcionario**: Responsável por armazenar os dados individuais (nome, cpf, cargo, salário) e gerenciar ações como reajuste e demissão.[cite: 1]
* **Departamento**: Gerencia o agrupamento de funcionários em um array fixo, controlando adições e somas salariais.[cite: 1]
* **TesteSistema**: Classe principal que executa o roteiro de testes obrigatório para validação da lógica.[cite: 1]
